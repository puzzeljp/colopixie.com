OS X El Capitan で ESET Cyber Security を使う

ウイルス対策ソフトウェアとしてWindows/Android/Macで５ライセンス使える「ESET」を使っていたのですが、El Capitan にアップデートしたらエラーがでて使えなくなってしまいました。

公式サイトへ行ってみると、

※ 現在は、ベータ版 / パブリックベータ版を含め、非対応となります。
ESET製品をご利用の端末については、OS X v10.11 El Capitan へのバージョンアップをお控えくださいますようお願いします。

OS X v10.11 El Capitan への対応について | ESETサポート情報

とのことで、使用できないようでした。
ただ、１１月中旬には新しいバージョンが公開されるようですが、今のところは公開されていないので困りました。

海外版は対応している模様

色々探して見ると、海外のESETのダウンロードページに、El Capitan対応の文字が。
Download ESET Cyber Security Pro for Mac | ESET
Ready to download?からダウンロードできました。

ダウンロードした dmg をダウンロードしたら、日本版では、ユーザ名とパスワードを要求されますが、海外版ではシリアルキーを求められました。
試しに、入れてみましたがきちんと認証されました。



インストール完了です。


エラーなく使えるようです。

El Capitanは色々とrootless や Photoshop の不具合等ありますが、ライブ日本語変換（使ってないですが）や、フォントが増えた事、ウィンドウの切り替え等のレスポンスの速さなど便利になった部分がありますね。
ソフトウェアは大体動くと思いましたが、rootless で制限されたフォルダ等にアクセスするソフトウェアについては対応できてないものもありますので、対応OSはきちんと見ておいたほうが良さそうです。
