K2100M と Windows 10 の組み合わせで画面描写がバグる対処方法

最近、QUADRO の K2100M が搭載されたＰＣを買って、プリいインストールは Windows 7 だったのですが、ディスプレイが4KでDPIの設定がうまくできなく重い腰を上げながら（？）
Windows 10 にアップデートしました。

そこで問題が。
動画やPhotoshop・Illustrator、Explorerでの画面描写がおかしなことになってしまいました。
少し負担をかけると下記のような状態に。

デスクトップの壁紙がちらちらと浮き出てくる現象。
謎ですし、映像を見るきが起きません。。。

そういえば、Chromeってなんかしてたよねってことで、chrome://flags/ にいき、DirectWrite を無効にするを有効にしてみました。
そうすると、Chromeで動画を再生しているときにはこのちらちら現象がなくなりました。

しかし、Explorerなどでは状況は変わりません。Windows Updateやメーカー提供のドライバーは降ってきません。。。

そこでシステムの復元とシステムバックアップを取り、NVIDIA から最新のドライバをインストールすることにしてみました。
Drivers – Download NVIDIA Drivers

インストールをし、再起動を行うと。。。あのチラチラが消えていきました！
Windows 10になり、裏側が多少なりとも変わったようです。

ただ、PhotoshopとIllustratorは３０秒毎に応答なしとなってしまうので、これはどうにかならないかと探っているところです。。。。とほほ。
