無刻印キーボードの VAIO Z

ついにきましたよー
無刻印キーボードの VAIO Z !
今回買ったのはこれ：VAIO | 商品情報 | VAIO Z
VAIO Z は、VAIO Z1 (Zシリーズ | | ソニー), VAIO Z21 (Zシリーズ | | ソニー)を持っていました。

スペック

  • 第二世代ハイスピードSSD（NVMe対応） 256GB
  • Windows 10 Pro
  • ブラック・勝色ダブルアルマイト仕様
  • VJZ1311A
  • Core i7-6567U（3.30GHz）
  • メモリー16GB（オンボード）
  • 13.3型ワイド（16:9） WQHD 2560×1440
  • 英字配列無刻印キーボード

 

じゃじゃーん！VAIOの箱です

MacBook と違いビニールではなく、不繊維の袋に入ってます

VAIO 社製の VAIOです。

勝色ダブルアルマイト仕様です。


購入カードが入ってました。

無刻印キーボードなので、キーボードが印刷された紙が入ってました。

本体です


VAIO Z ｷﾀ━━━━(ﾟ∀ﾟ)━━━━!!

で、でたー！！

Z ENGINE® 基板アクセサリーももちろんついてました。かっこいいですね。


モバイルメイン機の買い替え

MacBook Pro 15 inch with Retina Display (Early 2013) から VAIO Z 2016 に Mac から Windows に変更しましたが、最近ではLightroom、Photoshop、Illustrator は主に使ってますが、Vegas Pro も使うことも多いので Mac 上で Windows で使っていたので、そういった意味でWindowsでも心配なさそうです。
Sketch とか Flinto については MacBook があるのでそっちで代用しようかなと。

WIN SCORE SHAREで見ると、SSDの高さがずば抜けていますね。

Read が 2133 (MB/s) というノートPCでようわからん速さがでてます。ファイルのコピーだったりもめちゃくちゃ早いです。

Badなこと

  • なぜかWi-Fiがくっそ不安定です。いきなりWi-FiがOFFになり再起動しないかぎり復帰しません
  • お値段が高すぎます。
  • ブラックスクリーンになることが多いです。（スリープ解除したらブラックスクリーンでなにもできない・・・アプリの問題な気もする）
  • Vegas Pro + After Effects を同時起動すると、映像の音声が途切れ始める。Windowsのメインマシンが、Corei7 4810MQ / 32GB / K2100M とかいうノートPCなので、それと比べるとグラフィックとCPU性能は低くて辛さを感じます。CPUが追いつきません。
  • USBポートが2つしかない
  • Type C とかやっぱりほしい。
  • 外部ディスプレイがHDMIだけなのでも心もとない。

まとめ

ビジネスPCとしては、めちゃくちゃいいです。かっこいいし、Windows だからWordとかもめちゃ使えるし。SSD早すぎて、いろんなファイルが爆速で開きます。Photoshop,Illustratorもフォント数が5000くらいあってもさっと起動します。良いです。
ただ、After Effects とか C4D とかは流石に限界を感じます。辛いってなります。CPU が Quad Core じゃないとかいろいろありますが、仕方ないので諦めます。グラボに比較的負担をかけると、ブルースクリーンにもなるのも嫌ですね。

キーボードの打ち心地も上場で、このブログを書いている感じでは、普段使っている HHKB Pro よりは劣りますが、 MacBook Pro のキーよりは打ちやすく感じます。MacBook よりはすごくマシです。

クラムシェルが出た VAIO Z 、ぜひいかがですか。無刻印VAIO Z 所有感的には最高です。
VAIO S11 のインターフェイスで VAIO Z がでたらまた考えます。
